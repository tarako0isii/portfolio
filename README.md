## --作品--

### 投票アプリ
**概要**  
ペットに「いいね」を押すと、投票数がデータベースに保存され、
結果がバーグラフで表示されるWebアプリ。

**制作背景**  
FlaskとSQLiteを使って、データベースと連動する
Webアプリを自分で一通り実装してみたかったため。

**使用技術**  
- バックエンド：Python / Flask  
- データベース：
  - SQLite（開発環境）
  - PostgreSQL（Neon / 本番環境）
- フロントエンド：HTML / CSS  
- デプロイ：Vercel

**工夫した点・苦労した点**  
- POSTリクエストで投票数を更新する処理を実装した  
- データベース設計（tasksテーブル）を自分で考えた 
- 誰でも使用できるように公開環境を整えた
